Simple Web Calculator

A clean, responsive, and simple calculator web application built with HTML, CSS, and vanilla JavaScript. This project features a sleek dark mode interface and uses localStorage to remember your last calculation, even after you refresh the page.

That's it! The calculator will be running locally on your machine.
